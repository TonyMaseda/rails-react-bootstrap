# Rails React Bootstrap

Example application


## System dependencies, Frameworks, Libraries, Packages and Gems

* Ruby 2.7.2
* Ruby on Rails 6.0.3.4
* PostgreSQL
* React
* jQuery
* Bootstrap
* Font Awesome
* chartkick (chart js) + groupdate
* will_paginate
* Action Text (trix)
```
$ ruby -v
$ rails -v
$ rails new PROJECT_NAME -d=postgresql --webpacker=react
```


## Initialization

```
$ bundle install
$ yarn install
$ rails db:create
$ rails db:seed
$ rails s
```


## Authentication

* devise
* rolify
* cancancan


## Testing (rspec)

```
$ rspec
$ open coverage/index.html
```
